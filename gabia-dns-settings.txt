# Gabia DNS 설정 변경 가이드

## 1. Gabia 관리자 페이지 접속
- URL: https://www.gabia.com
- 로그인 후 "도메인 관리" → "DNS 관리" 이동

## 2. 각 도메인별 DNS 서버 변경

### 소셜리티.co.kr
현재 DNS 서버를 AWS Route 53으로 변경:
- DNS 서버 1: ns-xxx.awsdns-xx.com (Route 53에서 확인)
- DNS 서버 2: ns-xxx.awsdns-xx.co.uk
- DNS 서버 3: ns-xxx.awsdns-xx.org
- DNS 서버 4: ns-xxx.awsdns-xx.net

### sociality.co.kr
- 동일한 NS 레코드 사용

### sociality.kr
- 동일한 NS 레코드 사용

## 3. DNS 변경 후 확인사항
- DNS 변경 후 24-48시간 전파 시간 필요
- 변경 전에 Route 53 호스팅 영역이 생성되어 있어야 함
- NS 레코드를 정확히 복사해서 입력해야 함

## 4. 주의사항
- DNS 서버 변경은 기존 이메일, 서브도메인 등에 영향을 줄 수 있음
- 변경 전에 백업을 권장
- 비즈니스 시간에 변경하는 것을 권장
